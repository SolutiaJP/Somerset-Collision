To enable JavaScript in Internet Explorer
  1. Go to Tools > Internet Options > Security
  2. Highlight the Internet icon and click on the Custom Level button. In the Reset To box, select Medium and click Reset, Click OK to close this box, and OK again to close the main Internet Options box.
  3. Please close and reopen Internet Explorer and navigate to the page you were trying to reach.
To enable JavaScript in Firefox
  1. Go to Tools > Options > Content and click on Enable JavaScript.
  2. Click OK to close this box.
  3. Please close and reopen Firefox and navigate to the page you were trying to reach.
To enable JavaScript in Google Chrome
  1. Go to Customize and control Google Chrome and select Options.
  2. In the Google Chrome Options tab select the Under the Hood menu item.
  3. In the Privacy section click Content settings... button.
  4. In the Content settings window go to JavaScript section and select Allow all sites to run JavaScript (recommended).
  5. Close the Google Chrome Options tab.
  6. Click on the Reload this page button of the web browser to refresh the page.
To enable JavaScript in Safari
  1. On the web browser menu click on the Edit and select Preferences.
  2. In the Preferences window select the Security tab.
  3. In the Security tab section Web content mark the Enable JavaScript checkbox.
  4. Click on the "Reload the current page" button of the web browser to refresh the page.
